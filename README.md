# Something Fishy

Unity-based fishing and market simulation game built around catching fish, inventory management and progressing through the world.

## Structure

```text
something-fishy-v1/
├── Assets/
│   ├── Scenes/
│   ├── Scripts/
│   │   ├── Container/
│   │   ├── Data/
│   │   ├── Enviornment Manager/
│   │   ├── Fishing/
│   │   ├── Game Manager/
│   │   ├── Item Dictionary/
│   │   ├── Minigames/
│   │   ├── Player/
│   │   ├── Seagull/
│   │   └── UI/
│   ├── Settings/
│   └── TextMesh Pro/
├── Packages/
├── ProjectSettings/
├── UIElementsSchema/
├── SomethingFishyV1.slnx
└── something-fishy-v1.slnx
```

## GamePlay features

- Main meny with **New Game** and **Load Game**
- Fish spawning and interaction
- Fishing minigame / catch loop
- Inventory management and equipment
- Marketplace interactions for buying / selling
- Zone unlocking and progression
- Dynamic market changes
- Local save/load support

## Tech Stack

- C#
- Unity
- Unity Input System
- UI Toolkit
- TextMeshPro
- Newtonsoft Json
- Entity Framework Core

## Run Locally

### Option 1: Run from Unity

1. Open the project in Unity Hub
2. Use the correct Unity version for the project
3. Open the main gameplay scene
4. Press Play

### Option 2: Run from relase

1. Download from the latest release build
2. Extract the files
3. Launch the executable
